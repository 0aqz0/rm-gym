# rm-gym

RoboMaster AI Challenge Simulation Environment based on [OpenAI gym](https://github.com/openai/gym)

## Installation

1. Install gym

```bash
pip install gym
```

2. Download this repo and install this package via

```bash
pip install -e .
```
